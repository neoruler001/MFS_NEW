## 하네스: MFS (모바일 법인카드 관리 시스템)

**목표:** FastAPI 백엔드 + Vue 3 프론트엔드 풀스택 개발 작업을 에이전트 팀으로 조율하여 표준을 준수하며 구현한다.

**트리거:** MFS 기능 구현, API 추가/수정, Vue 뷰 개발, 버그 수정, 기능 개선 등 개발 작업 요청 시 `mfs-orchestrator` 스킬을 사용한다. 단순 질문이나 개념 설명은 직접 응답 가능.

**사용자 호칭:** Neo
